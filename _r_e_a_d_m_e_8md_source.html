1 # Optimized TM1637 library for Arduino
2 [![Build Status](https://travis-ci.org/Erriez/ErriezTM1637.svg?branch=master)](https://travis-ci.org/Erriez/ErriezTM1637)
3 
4 This is a 2-pin serial TM1637 chip library for Arduino, optimized for size and speed. It supports a combined LED driver controller and key-scan interface to detect one key press.

9 ## Chip features
10 
11 - Power CMOS process
12 - Display mode (8 segments Ã— 6 digits), support common anode LED output
13 - Key scan (8 x 2-bit), enhanced anti-jamming button recognition circuit
14 - Brightness adjustment circuit (adjustable duty cycle 8)
15 - Two-wire serial interface (CLK, DIO)
16 - Oscillation mode: Built-in RC oscillator
17 - Built-in power-on reset circuit
18 - Built-in automatic blanking circuit
19 - Package: DIP20 / SOP20

22 ## Hardware
23 
24 Connect power and 2 data pins to an Arduino board DIGITAL pins:
25 * VDD (Power 3.3V - 5V)
26 * GND (Ground)
27 * CLK (Clock)
28 * DIO (Bi-directional data input/output)
29 
30 The following TM1637 pins should be connected to LED's and buttons in a matrix:
31 * K1~K2 (Key-scan data input to read one key press after each other)
32 * SEG/GRID (Output for LED matrix)

44 * Check maximum regulator / diode current to prevent a burnout when using lots of LED's. Some boards can provide only 100mA, others 800mA max.

47 ## Two-wire serial interface
48 
49 The TM1637 communicates with a MCU serial by using two wires:
50 
51 * DIO (bi-directional input/output pin)
52 * SCL (Clock pin)
53 
54 **Note:** The serial interface is not compatible with I2C or TWI, because no device address with read/write bit is used.

69 ## Usage
70 
71 **Initialization**
72 
73 ```c++
74 // Include TM1637 library
75 #include "TM1637.h"
76 
77 // Connect display pins to the Arduino DIGITAL pins
78 #define TM1637_CLK_PIN 2
79 #define TM1637_DIO_PIN 3
80 
81 // Create tm1637 object
82 TM1637 tm1637(TM1637_CLK_PIN, TM1637_DIO_PIN);
83 
84 void setup()
85 {
86  // Initialize TM1637
87  tm1637.begin();
88 }
89 ```
90 
91 **Display on/off**
92 
93 ```c++
94 // Turn display off
95 tm1637.displayOff();
96 
97 // Turn display on
98 tm1637.displayOn();
99 ```
100 
101 **Turn all LED's off**
102 
103 ```c++
104 // Turn all LED's off
105 tm1637.clear();
106 ```
107 
108 **Get keys**
109 
110 ```c++
111 // Get 8-bit key-scan
112 uint8_t keys = tm1637.getKeys();
113 ```
114 
115 **Write Byte to display register**
116 
117 ```c++
118 // Write segment LED's to the first display registers 0x00..0x0F with value 0x00..0xff to
119 // display numbers and characters. Just an example which depends on the hardware:
120 tm1637.writeData(0x01, 0x01);
121 ```
122 
123 **Write buffer to display registers**
124 
125 ```c++
126 // Creat buffer with LED's
127 uint8_t buf[] = { 0b10000110, 0b00111111, 0b00111111, 0b00111111, 0b00111111, 0b00111111};
128 
129 // Write buffer to TM1637
130 tm1637.writeData(0x00, buf, sizeof(buf));
131 ```
